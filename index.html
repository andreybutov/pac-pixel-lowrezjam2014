'Pac-Pixel'

Entry for #lowrezjam 2014, a 32x32 pixel game jam.

original 32x32 pixels



mirror of original - zoomed 10x

Details

'Pac-Pixel' is a derivative of Pac-Man, implemented at a resolution of 32x32 pixels.

The game is implemented in plain Javascript, with no additional dependencies.

The actual game plays in the small 32x32 game screen at the top. The larger 320x320 game screen is presented for the player's convenience. The larger game screen contains a copy of each rendered frame of the original game, scaled 10x, so it's easier to play the game.

Unlike the original game, Pac-Pixel features a single on-going level where your goal is to eat as many food pellets as you can before being caught by one of the ghosts.

New food pellets will randomly appear on the playing field.

New ghosts will regularly enter the playing field as well.

Use the arrow keys to move your pac-pixel character.

If you're interested in how the game was made, the full source code is available here.
